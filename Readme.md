朴实无华的查询界面，仅支持以one-api/new-api/... 等 OpenAI 标准格式的 api 测活，本代码由 GPT 辅助完成

- 本地运行

    - 安装环境

    ```
    pip install -r requirements.txt
    ```

    - 执行代码

    ```
    python app.py
    ```

    - 打开浏览器：http://localhost:5000

- serverless 部署
    - vercel
    - zeabur: 需要添加环境变量 ZBPACK_SERVERLESS 和 PORT
